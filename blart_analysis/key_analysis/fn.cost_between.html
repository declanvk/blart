fn cost_between(

    kind: InnerNodeKind,

) -> f64
Expand description

Calculate the cost for using an inner node of new_width preceded by a inner node of width prev_width with the given kind calculated against the given distribution.

This function allows for incrementally calculating the assignment cost by summing the cost between successive pairs of widths in the assignment.
