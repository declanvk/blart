Trait AsBytes

Source 
pub trait AsBytes {
    // Required method
    fn as_bytes(&self) -> &[u8] ⓘ;
}
Expand description

Any type implementing AsBytes can be decomposed into bytes.

The primary purpose of this trait is to allow different types to be used as keys on the crate::TreeMap and TreeSet types.
