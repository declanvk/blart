Expand description

Node that contains a single leaf value.

Fields§

§value: V

The leaf value.

§key: K

The full key that the value was stored with.

§previous: Option<NodePtr<PREFIX_LEN, LeafNode<K, V, PREFIX_LEN>>>

Pointer to the previous leaf node in the trie. If the value is None, then this is the first leaf.

§next: Option<NodePtr<PREFIX_LEN, LeafNode<K, V, PREFIX_LEN>>>

Pointer to the next leaf node in the trie. If the value is None, then this is the last leaf.
