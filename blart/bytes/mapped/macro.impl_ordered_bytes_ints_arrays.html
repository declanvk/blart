§Safety

  1. This macro must only be used with types $elem where size_of::<$elem> == <$elem as AsBytes>::as_bytes(&instance).len(). The types must also be fixed size, so all instance of $elem must have the same size.
  2. This macro must only be used with types that implement
ⓘ
impl BytesMapping<$elem> for $mapping {
    // ...
}
  1. $elem must also implement
ⓘ
unsafe impl OrderedBytes for Mapped<$mapping, $elem> {}
