Struct TaggedPointer

Source 
#[repr(transparent)]
pub struct TaggedPointer<P, const MIN_BITS: u32>(NonNull<P>);
Expand description

A non-null pointer type which carries several bits of metadata.

The MIN_BITS constant is used to ensure that any type that is used with this pointer will have sufficient alignment to carry the specified number of bits. As a result, we can cast safely because we know that the const-evaluation process & type-checking will ensure that the number of bits is sufficient.

pub const ALIGNMENT: usize

The ABI-required minimum alignment of the P type.

The maximum alignment of a rust type is 2^29 according to the reference. On 32-bit and 64-bit platforms that means that it is impossible for the POINTER_MASK to be zero, there must be at least 3 bit present.
